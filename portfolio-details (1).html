Medical Web App with Stripe, Real-Time Chat & Booking

A full-featured medical platform built using Laravel and Livewire, providing seamless interaction between doctors, patients, and admins. It supports secure appointment booking, online payments, real-time chat, and hospital listings.

Key Features:

  • Multi-role system (Admin, Doctor, Patient)
  • Real-time chat using Laravel Echo and broadcasting
  • Stripe integration for secure online payments
  • Appointment scheduling, payment history, and medical profile management
  • Mobile-responsive and built with best security practices
